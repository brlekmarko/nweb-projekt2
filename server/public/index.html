Drugi projekt



(Stored) Cross-site scripting (XSS)

Klikom na gumb otvoriti će se stranica na kojoj se mogu upisivati komentari, no u komentare se može ubaciti i JavaScript kod koji će se izvršiti kada netko drugi otvori stranicu

Ako označite checkbox, svi komentari će se sanitizirati, odnosno ukloniti će se svi HTML tagovi iz komentara





Sensitive data exposure

Klikom na gumb otvoriti će se stranica na kojoj će se prikazati lozinke korisnika (recimo da ih je napadač dobio korištenjem SQL injectiona)

Ako označite checkbox, prikazat će se lozinke iz baze koja sprema hashirane lozinke (SHA256), a inače lozinke iz baze koja ih sprema u plain textu
